Cluster State Generation

Simulate the generation of GHZ and 3×2 cluster states in tin-vacancy color-center modules. Explore how photon-mediated entanglement links atomic memories into a graph-state resource for quantum computation, with configurable Barrett-Kok success probabilities.

Congestion on a Repeater Chain

Study how entanglement generation and swapping compete for memory resources along a linear chain of quantum repeaters. Adjust chain length, memory capacity, Bell pair generation rates, and fidelity parameters to observe congestion dynamics in real time.

Entanglement Swapping Switch

A star-topology quantum switch manages entanglement requests between multiple clients. Configure per-pair request rates and watch the scheduler balance memory slots and backlog queues — implementing algorithms from Promponas et al. (2024).

State Explorer
